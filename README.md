# claude-plugin

약사 크리에이터를 위한 Claude Code 플러그인 모음.

## pharmacist-studio

약사 크리에이터 콘텐츠 파이프라인 자동화 플러그인.

| 커맨드 | 설명 |
|--------|------|
| `/수집` | 올리브영 베스트 + 쓰레드 화제 게시물 실시간 수집 |
| `/성분` | 전성분 텍스트·이미지를 식약처 기준으로 분석 |
| `/비교` | 복수 제품 성분·후킹 포인트·점수 비교표 |
| `/통합` | 수집→성분→비교→기획 전체 파이프라인 자동 실행 |
| `/힉스필드` | higsfield.com AI 이펙트 콘텐츠 제작 가이드 |
| `/디자인` | SNS 카드뉴스/피드 이미지 디자인 분석 |
| `/각본` | 후킹·본론·CTA 갖춘 30~60초 쇼츠 각본 + 촬영 암기 키워드 |

## 설치

Claude Code 플러그인 마켓플레이스 또는 로컬 경로로 설치합니다.

```
/plugin install pharmacist-studio
```
