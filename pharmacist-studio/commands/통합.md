주제: $ARGUMENTS

## 실행 순서

아래 5단계를 순서대로 자동 실행한 뒤, **최종 결과를 하나의 HTML 리포트로 생성·표시**한다.

### ① /수집
Chrome MCP로 올리브영 + 쓰레드에서 [주제] 관련 데이터 수집.

### ② /성분
수집된 베스트 상품 1~2개의 전성분 분석 (웹 검색으로 전성분 확보 후 식약처 기준 평가).

### ③ /비교
수집된 상위 제품 2~3개 비교표 출력.

### ④ 콘텐츠 기획 방향 도출
수집·성분·비교 결과를 종합해 약사 크리에이터 콘텐츠 기획안 작성.

### ⑤ /각본 (각본 자동 생성)
④에서 정한 **1순위 콘텐츠**로 30~60초 쇼츠 각본(후킹 후보 3개·본론·CTA·촬영 암기 키워드)을 작성한다.

---

## 최종 출력: HTML 리포트 생성·표시

5단계 결과를 종합해 **하나의 완성된 HTML 파일**을 `Write` 도구로 생성한다.

- **파일 경로:** 현재 작업 디렉토리에 `통합리포트_[주제]_[YYYYMMDD].html`
- 마크다운이 아니라 **그대로 브라우저에서 열리는 완결된 HTML 문서**로 만든다.
- 생성하면 미리보기 패널에 바로 표시된다. 경로도 안내하고, 필요 시 Chrome MCP `navigate`로 `file://` 경로를 열어 보여준다.
- 아래 템플릿의 클래스·구조를 **그대로 활용**해 시각화한다. 1.시장 / 2.성분 / 3.비교 / 4.기획 / 5.각본 5개 섹션을 모두 담는다.

### 디자인 원칙 (꼭 지킬 것)
- 섹션 제목은 `<h2><span class="num">N</span> 제목</h2>` 형태로 번호 뱃지를 붙인다.
- 표는 `.tbl`로 감싸 둥근 모서리·헤더 그라데이션·줄무늬를 살린다.
- 후킹 후보는 `.hooks > .hook`(추천 1개는 `.hook.best`), 점수는 `.score`, 태그는 `.chip`, 유효/주의는 `.good`/`.warn`.
- 핵심 수치(함량·점수·SPF 등)는 `.stat` 카드로 강조.
