입력: $ARGUMENTS

## 핵심 제약 사항
- higsfield.com은 **사진 1장 업로드 → Create 버튼** 만 존재
- **텍스트 프롬프트 입력란 없음** — 영어 프롬프트 입력 불가
- 결과물은 업로드한 사진의 구도·조명·배경에 100% 의존
- 따라서 가이드의 핵심은 **어떤 사진을 찍느냐**

## 입력 없을 시 (소크라테스 모드)
주제/제품명이 없으면 다음 2가지만 질문:
1. 어떤 제품/주제인가요?
2. 원하는 분위기가 있나요? (자연스러운 / 세련된 / 임팩트 있는)

## 실행 순서

### STEP 1: 이펙트 선정
입력 주제에 맞는 higsfield 이펙트 TOP 3 선정.

추천 이펙트 풀:
- 액체 질감: `liquid-metal`, `liquid-chrome`, `liquid-gold`
- 자연 어우러짐: `fish-dreamcore`, `flora-bloom`, `petal-storm`
- 반짝임: `diamond-dust`, `crystal-shatter`, `holographic-foil`
- 피부/뷰티: `skin-glow`, `porcelain-skin`, `dewy-glass`
- 임팩트: `cosmic-explosion`, `fire-burst`, `neon-pulse`

### STEP 2: 가이드 출력
아래 포맷으로 출력.

### STEP 3: 브라우저 자동 오픈
가이드 출력 직후, 추천 이펙트 1위의 페이지를 Chrome MCP로 오픈:
`navigate` → `https://higsfield.com/video-effects/[이펙트명-소문자-하이픈]`
