## 실행 규칙

- 제품명 1개만 입력 → /성분 스킬로 자동 전환
- "알아서 해줘" 입력 → 올리브영 베스트 기준 자동 선정 (Chrome MCP로 수집)
- 제품명 2개 이상 → 비교표 출력

## 비교 항목
각 제품에 대해 웹 검색 또는 Chrome MCP로 전성분 수집 후:
1. 핵심 기능 성분 (유효 농도 추정)
2. 1% 기준선 위치
3. 콘텐츠 후킹 포인트
4. 약사 종합 점수 (5점 만점)

## 💾 옵시디언 저장 (정돈된 노트)
출력과 **동시에** 결과를 전용 보관함에 보기 좋은 노트로 저장한다.
- **보관함:** `/Users/namhojin/Documents/Claude/Projects/약사스튜디오` (없으면 자동 생성)
- **저장 폴더:** `<보관함>/비교/` · **파일명:** `<주제·키워드>_<YYYYMMDD>.md` (공백·특수문자는 `_`)

`Write`로 아래 구조의 노트를 만든다 — Obsidian 속성(properties)·태그·콜아웃으로 깔끔하게:

```markdown
---
유형: 비교
주제: "<주제·키워드>"
생성일: <YYYY-MM-DD>
제품수: <N>
추천1위: "<제품명>"
태그:
  - 약사스튜디오
  - 비교
---

> [!note] 최종 추천
> <1위 제품과 이유 한 줄>

<결과 본문 — 위 출력 포맷 그대로>
```

- 프런트매터 값과 태그에 **핵심 키워드**를 채워 그래프뷰·검색에서 자동 연결되게 한다.
- 저장 후 노트 경로를 한 줄로 안내한다. 저장 실패해도 출력은 정상 진행한다.
