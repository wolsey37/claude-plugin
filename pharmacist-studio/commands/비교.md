입력: $ARGUMENTS

## 실행 규칙

- 제품명 1개만 입력 → /성분 스킬로 자동 전환
- "알아서 해줘" 입력 → 올리브영 베스트 기준 자동 선정 (Chrome MCP로 수집)
- 제품명 2개 이상 → 비교표 출력

## 비교 항목
각 제품에 대해 웹 검색 또는 Chrome MCP로 전성분 수집 후:
1. 핵심 기능 성분 (유효 농도 추정)
2. 1% 기준선 위치
3. 콘텐츠 후킹 포인트
4. 약사 종합 점수 (5점 만점)

## 💾 옵시디언 저장
출력과 **동시에** 위 결과(노션 마크다운)를 옵시디언 보관함에 노트로 저장한다.
- **보관함 경로:** `/Users/namhojin/Documents/약사스튜디오` (약사 콘텐츠 전용 보관함)
  (없으면 자동 생성)
- **저장 폴더:** `<보관함>/비교/` (없으면 생성)
- **파일명:** `<주제·키워드>_<YYYYMMDD>.md` (공백·특수문자는 `_`로)
- `Write` 도구로 결과 마크다운을 그대로 저장하고, 마지막에 저장 경로를 한 줄로 안내한다.
- 저장 실패해도 결과 출력은 정상 진행한다.
