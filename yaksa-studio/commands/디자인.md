## 분석 항목

### 1. 색상 분석 (6:3:1 법칙)
- **메인 컬러 (60%):** Hex 코드 + 쓰임새
- **서브 컬러 (30%):** Hex 코드 + 쓰임새
- **포인트 컬러 (10%):** Hex 코드 + 쓰임새
- 전체 분위기: (따뜻한/차가운/중성)

### 2. 레이아웃 분석
- 구도 유형: (중앙정렬 / 3분할 / 황금비율 / 비대칭)
- 여백 활용도: (여백 많음 / 보통 / 꽉 찬)
- 텍스트-이미지 비율

### 3. 폰트 스타일
- 헤드라인 폰트: (세리프/산세리프/손글씨) + 추정 굵기
- 본문 폰트: 스타일
- 한글/영문 혼용 여부

### 4. 전체 디자인 키워드
3~5개 (예: 미니멀, 모던, 클린, 럭셔리, 귀여움)
