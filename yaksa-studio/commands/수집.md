키워드: $ARGUMENTS

**반드시 Claude in Chrome MCP(`mcp__Claude_in_Chrome__*`)로 브라우저를 직접 열어 수집한다.** WebSearch나 web_fetch로 대체하지 않는다.

## 실행 순서

### STEP 1: 탭 준비
`tabs_context_mcp(createIfEmpty: true)`로 탭 ID 확보.

### STEP 2: 올리브영 수집
`navigate` → `https://www.oliveyoung.co.kr/store/search/getSearchMain.do?query=[키워드 URL인코딩]`

페이지 로드 후 `get_page_text`로 텍스트 수집:
- 베스트 상품명 + 가격 (상위 10개)
- 브랜드명
- 카테고리 트렌드 키워드

### STEP 3: 쓰레드 수집
`navigate` → `https://www.threads.com/search?q=[키워드 URL인코딩]&serp_type=default`

페이지 로드 후 `get_page_text`로 수집:
- 화제 게시물 상위 5개 (내용 요약)
- 반응 지수 (좋아요/공유 수치가 있으면 포함)
- 소비자 인사이트 키워드
